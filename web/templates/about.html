About rhasspy

rhasspy is a toolkit for developing custom voice assistants, hosted in the Home Assistant platform (even on a Raspberry Pi).

This project was originally inspired by Jasper, a voice assistant developed by students at Princeton. Like Jasper, rhasspy is open source and built entirely from free software and intended to run with off-the-shelf hardware. However, there are some important differences that set rhasspy apart from Jasper and other voice assistants (Alexa, Google Home, etc.):

  • Offline by Default
    • rhasspy is designed to run without an internet connection, and without requiring the user to sign up for any 3rd party services. The option is there, however, if better accuracy is needed (such as using the snowboy hotword detector).
  • Hosted in Home Assistant
    • rhasspy is not a standalone piece of software, but exists as a set of resuable Home Assistant components. This gives rhasspy access to hundreds of components out of the box, and lets users incorporate only the pieces of rhasspy they're interested in.
  • Modern Tools and Techniques
    • Jasper used pocketsphinx for keyword matching on a small set of user-defined keywords. rhasspy instead takes a (possibly large) set of annotated training phrases, and uses them to both generate a language model and train an intent recognizer. In order to accomodate flexible speech input, the generated language model is "mixed" with CMU's base English model.

Software

rhasspy uses the following libraries/tools:

  • Wake Word
    • pocketsphinx
    • mycroft-precise
    • snowboy
  • Speech to Text
    • pocketsphinx - speech to text engine
    • webrtcvad - voice activity detection
    • SRILM - language modeling toolkit
    • phonetisaurus - guess pronunciations
    • sox - covert audio formats
  • Intent Recognition
    • rasaNLU - intent recognizer
    • spaCy - natural language processing
  • Text to Speech
    • pyTTSX3
    • espeak
    • pico-tts
    • MaryTTS
  • Miscellaneous
    • flask - web server
    • pyhocon - configuration
    • PyAudio - recording audio
